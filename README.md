## About

This repository documents my journey of learning **Low Level Design (LLD)** using **Java**, following the **TakeUForward (TUF+) Low Level Design Course**.

The purpose of this repository is to build a strong understanding of software design by implementing every concept from scratch. Rather than simply learning design patterns, the focus is on understanding the problems they solve, the reasoning behind each design choice, and how they are applied in real-world systems.

Each topic is accompanied by visual diagrams and clean Java implementations to make the learning process more intuitive and practical.

---

## What's Included

Each design pattern in this repository contains:

- ☕ Java implementation
- 📐 UML Class Diagram
- 🎨 Excalidraw Diagram
- 🖼️ Class Diagram Image
- 📖 Clean and well-structured source code
- 💡 Practical understanding of the design

- Learn Low Level Design through practical Java implementations.
